  <action>
    In `certifications.ts`, modify the `ngOnInit()` method. The existing `addStructuredData` call (around line 46-61) adds a plain `ItemList`. Replace it with an `ItemList` that wraps `EducationalOccupationalCredential` items.

    Replace the existing `this.seoService.addStructuredData({...}, 'certifications')` call with:

    ```typescript
    const certs = this.certifications();
    const credentialSchemas = certs.map((cert) => ({
      '@type': 'EducationalOccupationalCredential',
      name: cert.name,
      credentialCategory: 'Professional Certificate',
      recognizedBy: {
        '@type': 'Organization',
        name: cert.issuer,
      },
      dateCreated: cert.issued_raw,
      url: cert.certificateUrl,
      competencyRequired: cert.skills.join(', '),
      holder: {
        '@type': 'Person',
        '@id': 'https://frxncismor.dev/#person',
        name: 'Francisco Moreno',
      },
    }));

    this.seoService.addStructuredData({
      '@context': 'https://schema.org',
      '@type': 'ItemList',
      name: isEnglish
        ? 'Professional Certifications - Francisco Moreno'
        : 'Certificaciones Profesionales - Francisco Moreno',
      description: isEnglish
        ? 'Professional certifications in web technologies, frontend development, and software engineering.'
        : 'Certificaciones profesionales en tecnologias web, desarrollo frontend e ingenieria de software.',
      url: 'https://frxncismor.dev/certifications',
      author: {
        '@type': 'Person',
        '@id': 'https://frxncismor.dev/#person',
        name: 'Francisco Moreno',
      },
      itemListElement: credentialSchemas.map((cred, index) => ({
        '@type': 'ListItem',
        position: index + 1,
        item: cred,
      })),
    }, 'credentials');
    ```

    Note: The `const certs = this.certifications()` call reads the computed signal. Since `ngOnInit` fires after the component is created and the computed depends on i18n (which initializes via APP_INITIALIZER before any component loads), this is safe.
